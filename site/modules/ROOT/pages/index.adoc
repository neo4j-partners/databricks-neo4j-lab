= Databricks + Neo4j: Production AI Agents with Graph and Lakehouse

== Welcome

This workshop teaches you to build a production architecture that combines Neo4j graph databases with Databricks AI/ML. You will work with an Aircraft Digital Twin dataset, loading data into Neo4j, adding semantic search, and building a multi-agent system that routes questions to the right data source.

== Architecture

image::lab-architecture-overview.png[Multi-agent supervisor routing to Genie and Neo4j MCP agents]

The workshop culminates in a **Supervisor Agent** built with Databricks Agent Bricks that routes questions to specialized subagents: a **Genie space** for sensor telemetry analytics (natural language to SQL) and a **Neo4j MCP Agent** for graph relationship queries (natural language to Cypher).

== Prerequisites

* A Databricks workspace (provided for the workshop)
* A Neo4j Aura instance (pre-provisioned by workshop administrators)
* Basic familiarity with SQL and Python
